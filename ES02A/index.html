La pagina è una Home Page semplice con un messaggio di benvenuto ("Benvenuto nella mia Home Page"). In fondo alla pagina c'è un link che porta l'utente a un'altra pagina chiamata "index.html", situata nella sottocartella "/ES02A". Questo link potrebbe essere per accedere a una pagina riservata.

Questo codice HTML crea una pagina di login con due campi obbligatori: uno per il nome utente e uno per la password. Quando l'utente preme il pulsante "Accedi", i dati vengono inviati tramite il metodo POST al server per essere elaborati dal file PHP login.php. Il form è stato progettato in modo che l'utente debba compilare tutti i campi prima di poter inviare il form, grazie all'attributo required.
